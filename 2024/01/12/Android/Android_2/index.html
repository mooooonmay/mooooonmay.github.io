kotlin 数据类型

基本数值类型

类型 最小值 最大值
Byte （8） -128 127
Short （16） -32768 32767
Int （32） -2,147,483,648 (-2^31) 2,147,483,647 (2^31 - 1)
Long （64） -2,147,483,648 (-2^31) 2,147,483,647 (2^31 - 1)
Int -2,147,483,648 (-2^31) 2,147,483,647 (2^31 - 1)

所有未超出 Int 最大值的整型值初始化的变量都默认为 Int 类型，
如果初始值超过了其最大值，那么推断为 Long 类型。在数字值后面显式添加 L 表示一个 Long 类型

浮点类型

类型 位宽
Float （8） 32
Double （16） 64

Kotlin 对于小数的默认推断是 Double 类型。如果需要显式将一个小数指定为 Float 类型需要在数值尾部加入 f 或 F。由于 Float 类型十进制位数是 6 位，所以下述例子中 floatNumber 实际值大小为 3.1415926，后面就会出现进度丢失舍弃。

在 Kotlin 中还有一点与 Java 不同的是，Kotlin 中数字不存在隐式的拓宽转换。比如一个函数参数为 Double 的函数只能接收 Double 类型，不能接收 Float、Int 或者其他数字类型

字符类型 Char

字符的值需要用单引号括起来: '0'、'9'。

布尔类型 Boolean

在 Kotlin 使用Boolean表示布尔类型，它只有两个值 true 和 false。注意可空类型Boolean?类型会存在装箱操作。

位运算

Kotlin 中的位运算和 Java 不同的是没有用特殊符号来表示，可以采用了中缀函数方式调用具名函数。

  • shl(bits) – 有符号左移【shl 是 Shift Logical Left 的缩写】
  • shr(bits) – 有符号右移
  • ushr(bits) – 无符号右移
  • and(bits) – 位与
  • or(bits) – 位或
  • inv() – 位非
  • xor(bits) – 位异或
